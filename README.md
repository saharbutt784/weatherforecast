# 🌦️ Weather Forecast App

A clean and responsive web application that provides real-time weather forecasts based on user location or city searches. 

## 🚀 Features
* **Real-time Data:** Fetches up-to-date weather details including temperature, humidity, wind speed, and weather conditions.
* **City Search:** Look up current weather conditions for any major city worldwide.
* **Responsive Design:** Fully optimized to look great on desktop, tablet, and mobile devices.
* **Dynamic Visuals:** Changing background themes or icons based on the current weather condition (e.g., sunny, rainy, snowy).

## 🛠️ Built With
* **HTML5** - Structured content
* **CSS3** - Custom styling and layout (Flexbox/Grid)
* **JavaScript (ES6+)** - Fetching API data and handling DOM manipulation
* **Weather API** - OpenWeatherMap API
